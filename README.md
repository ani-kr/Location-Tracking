# Location Tracking Application (Core PHP)

This application captures user details along with their exact geographical location (latitude and longitude) using the browser’s geolocation API and displays the location on Google Maps.


## Features

- User data submission form
- Automatic latitude and longitude detection
- Stores location data in MySQL
- Displays location on Google Maps
- Modern, clean UI


## Tech Stack

- Core PHP
- MySQL
- HTML5
- CSS3
- JavaScript (Geolocation API)
- Google Maps Embed
- XAMPP (Local Server)


## Database Structure

### Table Used
- `locations` – Stores user details and location coordinates


## How to Run

1. Install **XAMPP**
2. Start **Apache** and **MySQL**
